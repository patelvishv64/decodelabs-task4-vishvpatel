# Project 4: Data Visualization Dashboard

## Overview

This project focuses on transforming raw e-commerce sales data into meaningful visual insights using Microsoft Excel. The dashboard provides an interactive and professional view of business performance through charts, KPIs, and data analysis techniques.

## Objective

The primary objective of this project is to:

* Convert raw data into visual representations
* Identify trends and patterns
* Compare business metrics
* Support data-driven decision making
* Present insights in a clear and professional dashboard

## Dataset Information

The dataset contains e-commerce transaction records with the following fields:

* Order ID
* Date
* Customer ID
* Product
* Quantity
* Unit Price
* Payment Method
* Order Status
* Items in Cart
* Coupon Code
* Referral Source
* Total Price

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Visualization Techniques
* Conditional Formatting
* Dashboard Design

## Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

* Total Revenue
* Total Orders
* Unique Customers
* Average Order Value

## Visualizations Created

### 1. Revenue by Product

Bar chart showing revenue generated by each product category.

### 2. Monthly Revenue Trend

Line chart displaying revenue growth and seasonal trends over time.

### 3. Order Status Distribution

Pie chart illustrating the percentage of orders in different statuses.

### 4. Payment Method Analysis

Bar chart comparing customer payment preferences.

## Key Insights

* Revenue performance can be tracked across products.
* Monthly trends reveal seasonal fluctuations.
* Order status distribution helps monitor operational efficiency.
* Payment method analysis identifies customer preferences.
* Customer purchasing behavior can be analyzed through order data.
